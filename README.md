# 🏆 PRO TRADING SIGNALS - محطة التداول الاحترافية

> تطبيق ويب احترافي لإشارات التداول الحقيقية مع ربط أسعار مباشر وتحليل فني متكامل

---

## ✨ المميزات / Features

### 📊 المؤشرات الفنية المدمجة (8 مؤشرات)
| المؤشر | الوصف |
|--------|-------|
| **EMA 200** | فلتر الاتجاه الرئيسي |
| **RSI (14)** | مؤشر القوة النسبية - تشبع شراء/بيع |
| **MACD (12/26/9)** | زخم الاتجاه والتقاطعات |
| **ATR (14)** | متوسط المدى الحقيقي - الأهداف ووقف الخسارة |
| **VWAP** | المتوسط المرجح بالحجم |
| **Bollinger Bands** | نطاقات التقلب |
| **Supertrend** | مؤشر الاتجاه القوي |
| **Stochastic** | مذبذب الزخم |

### 🐍 استراتيجية الثعبان (Snake Strategy)
- تقاطع EMA9 مع EMA21
- تأكيد RSI (منطقة 45-55)
- فلتر EMA50 للاتجاه

### 🧠 Smart Money Concepts
- Order Blocks الصاعدة والهابطة
- تحديد مناطق السيولة
- تحليل هيكل السوق (HH/HL, LH/LL)

### 🎯 Pivot Points
- المحور الرئيسي
- مستويات الدعم S1/S2/S3
- مستويات المقاومة R1/R2/R3

### 🕯️ تحليل الشموع
- تحديد نمط الشمعة الحالية
- توقع اتجاه الشمعة القادمة مع نسبة نجاح
- أنماط: Marubozu, Hammer, Shooting Star, Doji

### 💰 إدارة الصفقات
- نقطة الدخول الدقيقة
- وقف الخسارة المحسوب بـ ATR (1.5x)
- 3 أهداف ربح (TP1, TP2, TP3)
- نسبة المخاطرة/المكافأة (R:R)

## 🚀 التثبيت والتشغيل

### المتطلبات
- Python 3.9+
- اتصال بالإنترنت

### تثبيت سريع (Windows)
```bash
# 1. تحميل المشروع
git clone https://github.com/aabdullah230200-ship-it/pro-trading-signals-.git
cd pro-trading-signals-

# 2. تثبيت المكتبات
pip install -r requirements.txt

# 3. تشغيل التطبيق
python app.py
```

### تشغيل على Linux/Mac
```bash
chmod +x run.sh
./run.sh
```

### فتح التطبيق
بعد التشغيل، افتح المتصفح على:
```
http://localhost:5000
```
